# 添加组件

当然，默认提供的组件可能满足不了你的需求，你也可以添加自己的组件

## 添加 my-table 组件

```js
// const { transformCss2Js } = require('@icestack/shared')
/**
 * @type {import('@icestack/ui').Config}
 */
const config = {
  outdir: './my-ui',
  components: {
    // 要添加的组件名称
    myTable: {
      // 选择器
      selector: '.my-table',
      schema: (opts) => {
        // selector 为选择器
        // types 为各种 primary success 这样的颜色变量
        // or use transformCss2Js exported by '@icestack/shared'
        const { selector, types } = opts
        return {
          selector,
          defaults: {
            base: {
              [selector]: {
                // 交给 tailwindcss 处理
                apply: ['relative w-full', 'rounded text-left text-sm'],
                css: {
                  display: 'table'
                },
                '.thead': {
                  css: {
                    display: 'table-header-group'
                  },
                  apply: ['whitespace-nowrap text-xs font-bold']
                },
                '.tbody': {
                  css: {
                    display: 'table-row-group'
                  }
                },
                '.tfoot': {
                  css: {
                    display: 'table-footer-group'
                  },
                  apply: ['whitespace-nowrap text-xs font-bold']
                },
                '.tr': {
                  css: {
                    display: 'table-row'
                  }
                },
                '.td,.th': {
                  apply: ['px-4 py-3 align-middle'],
                  css: {
                    display: 'table-cell'
                  }
                }
              }
            },
            utils: {
              [selector]: {
                [`&-xs`]: {
                  ':not(.thead):not(.tfoot) .tr': {
                    apply: 'text-xs'
                  },
                  '.th,.td': {
                    apply: 'px-2 py-1'
                  }
                },
                [`&-sm`]: {
                  ':not(.thead):not(.tfoot) .tr': {
                    apply: 'text-sm'
                  },
                  '.th,.td': {
                    apply: 'px-3 py-2'
                  }
                },
                [`&-md`]: {
                  ':not(.thead):not(.tfoot) .tr': {
                    apply: 'text-sm'
                  },
                  '.th,.td': {
                    apply: 'px-4 py-3'
                  }
                },
                [`&-lg`]: {
                  ':not(.thead):not(.tfoot) .tr': {
                    apply: 'text-base'
                  },
                  '.th,.td': {
                    apply: 'px-6 py-4'
                  }
                }
              }
            }
          }
        }
      }
    }
  }
}
module.exports = config
```

## transformCss2Js

`transformCss2Js` 作为一个方法，可以快速的把带有 `@apply` 解析成:

```ts
{
  [selector]:{
    css:{},
    apply:[],
    [childSelector]:{
      //..
    }
  }
}
```

所以使用这个方式可以辅助你编写 `schema`

## !important

假如你要使用 `!important`

```ts
// 渲染单行 @apply
apply: 'bg-gray-400 text-yellow !important'
// 渲染2行 @apply
apply: ['bg-gray-100 text-blue', 'bg-gray-400 text-yellow !important']
// css 节点
css: {
  color: 'red !important'
}
```
