## Installation and use

## Installation

First you must install the `nodejs` environment.

Then use `npm init` to create `package.json`, and then in the `package.json` directory, execute the command.

```sh
# npm
npm i -D @icestack/ui
# yarn
yarn add -D @icestack/ui
# pnpm
pnpm i -D @icestack/ui

```

`@icestack/ui` will be downloaded and installed locally, and a binary command `icestack` will be registered.
