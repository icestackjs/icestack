# 扩展组件尺寸形状样式

## 介绍

要想针对每个组件行为尺寸形状样式做一些修改，这里就必须使用到 `components` 配置项：

比如我们要给 `button` 添加 `2` 个尺寸 `xl`,`2xl`

```js filename="icestack.config.js"
/**
 * @type {import('@icestack/ui').Config}
 */
const config = {
  outdir: './my-ui',
  components: {
    // 组件名称
    button: {
      extend: {
        utils: [
          // Sizes
          // add xl
          `.btn-xl {
        @apply min-h-20 h-20 px-8;
        font-size: 1.25rem;
      }`,
          // add 2xl
          `.btn-2xl {
        @apply min-h-24 h-24 px-10;
        font-size: 1.5rem;
      }`
        ]
      }
    }
  }
}
module.exports = config
```
