# Command Line Interface

## Usage

```bash
icestack [command]
```

## Order

| Options | |
| ------- | ------------------------------- |
| `init` | Initialize the configuration file in the current directory |
| `build` | Read the configuration and build the style product |
| `watch` | Monitor configuration file changes and build styles immediately |

## icestack init

Initialize the configuration file in the current directory

### --only-config

Only initialize the configuration file, do not modify the `.gitignore` file, and run `build` to build

### --format \<format\>

**Type:** 'cjs','esm','ts'

Format for creating configuration files

## icestack build

Read the configuration and build the style product

### --clean

Whether to clean `outdir`

### -c, --config \<path\>

Loaded configuration path

## icestack watch

Monitor configuration file changes and build styles immediately

### -c, --config

Loaded configuration path
