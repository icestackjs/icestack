# 在微信小程序里使用

`@icestack/ui` 内部实现了一套预设来兼容微信小程序

你可以使用 `@icestack/ui/presets` 中导出的 `miniprogramPreset` 来构建出小程序版本的默认 `ui` 库

```js
const { miniprogramPreset } = require('@icestack/ui/presets')
/**
 * @type {import('@icestack/ui').Config}
 */
const config = {
  // 可以根据环境变量，把 h5 和 小程序的 outdir 设置为不同的目录
  outdir: './my-ui',
  presets: [miniprogramPreset()]
}

module.exports = config
```

这样你执行:

```bash
npx icestack build
```

的时候，构建出的就是适配小程序的库了。
