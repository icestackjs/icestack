# 安装及使用

## 安装

```sh
npm i -D @icestack/ui

yarn add -D @icestack/ui

pnpm i -D @icestack/ui
```

## 使用方式

主要有以下几种:

- [作为 tailwindcss plugin 来使用](/?path=/docs/文档-安装及使用-tailwindcss-plugin)
- [以 Cli 命令行使用](/?path=/docs/文档-安装及使用-cli)
- [Nodejs API 调用](/?path=/docs/文档-安装及使用-nodejs-api)
