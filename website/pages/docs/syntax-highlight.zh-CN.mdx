# IDE语法高亮

在编写 `css schema` 或者编写 `extend` 配置时，往往苦于 `js` 字符串没有 `css` 高亮：

![](/syntax-highlight/before.png)

我们往往需要 `css/scss` 智能提示与代码高亮，此时可以遵循一下方式获得。

## vscode

安装 `vscode-styled-components` 扩展

然后使用

```js
const { css } = require('@icestack/ui')
// or
const css = String.raw
```

然后使用这个方法包裹一下，就有高亮提示了:

![](/syntax-highlight/after.png)

## tailwincss 提示

添加 `.vscode/settings.json`, 内容如下所示:

```json
{
  "tailwindCSS.experimental.classRegex": ["css`([^`]*)`", "@apply([^`;]*)"]
}
```

然后在同级目录下，新建 `tailwind.config.js`, 这样 `@apply` 的智能提示就有了。

![](/syntax-highlight/hint.png)
