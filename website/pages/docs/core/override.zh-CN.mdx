# 扩展组件尺寸形状样式

## 介绍

要想针对每个组件行为尺寸形状样式做一些修改，这里就必须使用到 `components` 配置项：

比如我们要给 `button` 添加 `4` 个尺寸 `xl`,`2xl`,`3xl`,`4xl`

我们可以使用下列的写法（推荐直接使用 Css 字符串或一个函数返回字符串）

```js filename="icestack.config.js"
/**
 * @type {import('@icestack/ui').Config}
 */
const config = {
  outdir: './my-ui',
  components: {
    // 组件名称
    button: {
      extend: {
        utils: [
          // Sizes
          // xl
          `.btn-xl {
        @apply min-h-20 h-20 px-8;
        font-size: 1.25rem;
      }`,
          // 2xl
          {
            '.btn-2xl': {
              apply: 'min-h-24 h-24 px-10',
              css: {
                'font-size': '1.5rem'
              }
            }
          },
          // 3xl
          ({ selector }) => {
            return {
              [`${selector}-3xl`]: {
                apply: 'min-h-28 h-28 px-12',
                css: {
                  'font-size': '1.75rem'
                }
              }
            }
          },
          // 4xl
          ({ selector }) => {
            return `
        ${selector}-4xl {
          @apply min-h-32 h-32 px-14;
          font-size: 2rem;
        }
        `
          }
        ]
      }
    }
  }
}
module.exports = config
```
