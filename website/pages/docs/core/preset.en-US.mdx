## Refining and Using Presets

## Introduction

Presets act as a type of configuration, you can distill your configuration into a preset

```js filename="icestack.config.js"
/**
 * @type {import('@icestack/ui').Config}
 */
const config = {
  outdir: './my-ui',
  presets: [myPreset, myFunctionPreset(options)]
}
module.exports = config
```
