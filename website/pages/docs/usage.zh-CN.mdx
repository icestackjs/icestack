# 安装及使用

## 安装

首先你必须安装好 `nodejs` 环境:

然后使用 `npm init` 创建 `package.json`，再在 `package.json` 目录下，执行命令:

```sh
# npm
npm i -D @icestack/ui
# yarn
yarn add -D @icestack/ui
# pnpm
pnpm i -D @icestack/ui
```

`@icestack/ui` 将会被下载并安装在你的本地，同时一个二进制命令 `icestack` 将会被注册
