# Command Line Interface

## 使用方式

```bash
icestack [command]
```

## 命令

| Options |                                |
| ------- | ------------------------------ |
| `init`  | 在当前目录初始化配置文件       |
| `build` | 读取配置，并构建样式产物       |
| `watch` | 监视配置文件更改，立刻构建样式 |

## icestack init

在当前目录初始化配置文件

### --only-config

只初始化配置文件，不修改 `.gitignore` 文件，并进运行 `build` 构建

### --format \<format\>

**Type:** 'cjs','esm','ts'

创建配置文件的格式

## icestack build

读取配置，并构建样式产物

### --clean

是否清理 `outdir`

### -c, --config \<path\>

加载的配置路径

## icestack watch

监视配置文件更改，立刻构建样式

### -c, --config

加载的配置路径
